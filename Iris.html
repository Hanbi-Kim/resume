2.2 Neighbour selection of KNN method¶

Let's increase the accuracy by choosing different N!¶

Since we have 120 observations in train set,it means that each observation have 119 neighbour around. The number of nearest neighbour is likely to be selected from 1 to 119, however, we have three different species in this case. Thus, 40 will be the maximum range we will see.

Final Conclusion:¶


our data set contains 150 Irises in total with sepal length and width in cm, and also petal length and width in cm. Each iris observations have information about their speciesc as well. Therefore, all the 150 irises in data set is divided fairly into three different species with the same number,50, namly, setosa, virsicolor and virginica. Our goal is running several machine learning methods to classify iris, and find the best method.

Firstly, we need to arrange our date set before we start any analysis. That means, we need to find any missing values or some typos as well. First few steps show that our data set is already managed clearly without any missing values and it seems like every observed values are quite well-distributed.

Visualizing data is helpful and sometimes it is crucial. Since it is not possible to draw high dimension plots, I divided each observations as petal and sepal section and plotted them as two different scatter plots. If we look at scatter plot of sepal lengh and width, Iris-setosa is clearly separated away from the other two species,but it is not clear enough to say for the other two species. However if we move to petal section, it becomes much clearer and three species seem like they have thier own positions.

To find the best method, we need to test them and verify whether their modelling formula is better than other methods. Thus, we will split our data set into test and training data set. We will use the traning set as fitting model with selected machine learning method, and test set as testing them whether thier model fits well in test set.

At last, both SVM method with linear kernel and logisitic regression show 100% corredcted prediction after testing. Even so, other methods also show high accuracy with at least 80%. So, the four different variables of length and width of sepal and petal can be used as characteristics to classify these three iris species.
